# Linux System Health Check Tool 🐧

A Bash-based system monitoring tool that generates a structured health report containing important Linux system information.

This project is part of my Cloud Infrastructure Journey, where I am building practical automation and infrastructure skills.

---

## 📌 Project Overview

System administrators and DevOps engineers regularly monitor servers for:

- System information
- CPU resources
- Disk utilization
- Memory usage
- Running processes
- Environment details

This script automates basic health checks and generates a readable system health report.

## ⚙️ Features

### System Information
Collects:

- Current user
- Kernel version
- Operating system
- Date and time

### CPU Monitoring

Displays:

- Available CPU cores

### Disk Monitoring

Shows:

- Filesystem usage
- Disk capacity
- Available storage

### Memory Monitoring

Checks memory availability and handles unsupported environments.

### Process Monitoring

Displays currently running processes.

### Report Generation

Automatically creates:
reports/health_report.txt
